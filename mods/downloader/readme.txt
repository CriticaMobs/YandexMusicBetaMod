Позволяет скачивать треки по его id. Без токена работать не будет, апи вернет только 30 секунд демо трека.
Запрос подписывается HMAC SHA-256 хэшем в base64 кодировке. Ключ для хмака хранится в исходниках приложения, там есть ключи для виндовых и маковых билдов.

Скачивание работает в 2х режимах - в альбоме (скачивает весь альбом) и в плеере (скачивает текущий трек)
